All commands run on Glass via SSH.

1. Configure StopSpam (or equivalent anti-spam plugin):

   The exact config keys depend on which anti-spam plugin was installed. Read the config file first.

   Key settings to configure (per D-08):
   - Message cooldown: 5 seconds between messages from the same player
   - Similarity detection: enabled (block near-duplicate messages)

   For StopSpam, typical config.yml settings:
   ssh glass "cat > /opt/hermescraft/server/plugins/StopSpam/config.yml << 'EOF'
# StopSpam Configuration
enabled: true
cooldown: 5
similarity-check: true
similarity-threshold: 0.8
max-caps-percentage: 80
block-repeated-messages: true
warn-message: '&cPlease wait before sending another message.'
EOF"

   If the plugin uses different config keys, adapt accordingly based on what read_first reveals.
   Key principle: 5-second cooldown, similarity detection ON.

2. Configure ServerTap:
   ServerTap should expose a REST API on port 4567 by default.

   Check if ServerTap config needs the port to be exposed through Docker:
   The current Docker run command only exposes port 25565. ServerTap port 4567 needs to be accessible.

   Option A: Access ServerTap from INSIDE the Docker container (sufficient for now if agent runs on same host).
   Option B: Re-create Docker container with additional port mapping.

   For now, check if ServerTap is accessible:
   ssh glass "docker exec mc-server curl -s http://localhost:4567/v1/server 2>/dev/null || echo 'ServerTap not responding inside container'"

   If it works from inside the container, that's fine for now. The agent can access it via docker exec or we can add port mapping later.

   If it needs external access, re-create the container:
   ssh glass "docker stop mc-server && docker rm mc-server"
   ssh glass "docker run -d --name mc-server -p 25565:25565 -p 4567:4567 --memory=3g -v /opt/hermescraft/server:/data -w /data eclipse-temurin:21-jre java -Xmx3G -Xms2G -jar paper-1.21.1.jar nogui"
   Wait 30-60 seconds for startup.

   Then verify:
   ssh glass "curl -s http://localhost:4567/v1/server 2>/dev/null | head -5"

3. Read ServerTap config to check API key requirement:
   ssh glass "cat /opt/hermescraft/server/plugins/ServerTap/config.yml 2>/dev/null"
   If an API key is required, note it. If authentication can be disabled for local access, disable it.

4. Restart server to apply StopSpam config:
   ssh glass "docker restart mc-server" && sleep 30

5. Final plugin verification — check all plugins from both batches (02 + 03) are loaded:
   ssh glass "docker logs mc-server 2>&1 | grep -i 'Enabling' | sort"
