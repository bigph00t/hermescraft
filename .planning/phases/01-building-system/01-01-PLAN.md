1. Create directory `agent/blueprints/` for JSON blueprint files.

2. Create `agent/blueprints.js` — the blueprint loader module. Follow project conventions: camelCase exports, ES modules, one-line file header comment, no default exports.

   Exports:
   - `loadBlueprint(name)` — reads `agent/blueprints/{name}.json`, parses, validates, returns blueprint object or null
   - `listBlueprints()` — returns array of available blueprint names (filenames without .json)
   - `resolvePalette(blueprint, availableBlocks)` — given a blueprint and an array of block names the agent has, returns a mapping from palette slot to actual block (e.g., "wall" -> "oak_planks"). Uses the blueprint's palette preferences but falls back to whatever the agent has.
   - `getBlueprintMaterials(blueprint)` — returns a Map of block_name -> count_needed (total blocks required)

   Blueprint JSON format (per research — layer-by-layer, relative coordinates):
   ```json
   {
     "name": "small_cabin",
     "description": "A 5x5 wooden cabin with door and flat roof",
     "size": { "x": 5, "y": 5, "z": 7 },
     "palette": {
       "W": { "preferred": ["oak_planks", "spruce_planks", "birch_planks"], "tag": "wall" },
       "L": { "preferred": ["oak_log", "spruce_log", "birch_log"], "tag": "pillar" },
       "F": { "preferred": ["oak_planks", "cobblestone", "stone"], "tag": "floor" },
       "R": { "preferred": ["oak_slab", "spruce_slab", "cobblestone_slab"], "tag": "roof" },
       "D": { "preferred": ["oak_door"], "tag": "door" },
       "G": { "preferred": ["glass_pane", "glass"], "tag": "window" }
     },
     "layers": [
       {
         "y": 0,
         "comment": "Foundation/floor",
         "grid": [
           "FFFFF",
           "FFFFF",
           "FFFFF",
           "FFFFF",
           "FFFFF"
         ]
       }
     ]
   }
   ```

   The grid is a 2D string array per Y layer. Each character maps to a palette key. Space or "." means air (skip). Grid[row] is Z-axis (row 0 = z=0), each character in the string is X-axis.

   `resolvePalette(blueprint, availableBlocks)` logic:
   - For each palette entry, iterate through `preferred` list
   - Return the first preferred block that appears in `availableBlocks`
   - If none match, return the first preferred block (agent will need to gather)
   - Return object: `{ "W": "oak_planks", "L": "oak_log", ... }`

   `getBlueprintMaterials(blueprint)` logic:
   - Iterate all layers, all grid rows, count each non-air palette character
   - Return Map of resolved block names to count

3. Create `agent/blueprints/small-cabin.json` — a 5x5 house satisfying BUILD-01:
