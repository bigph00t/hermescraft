Otherwise try `readFileSync(BRAIN_STATE_FILE, 'utf-8')` + `JSON.parse` in try/catch. On success, update cache and return `formatBrainState(parsed)`. On any error (ENOENT, parse), return `null`. Never throws.

**Internal functions (not exported):**

3. `function pushRingBuffer(arr, item, maxSize)` — Push item, while length > maxSize do `arr.shift()`, return arr.

4. `function writeBrainState(state)` — Apply ring buffer caps (insights:20, spatial:50, partnerObs:100) via pushRingBuffer-style while loops. `JSON.stringify(state, null, 2)` -> `writeFileSync(BRAIN_STATE_TMP, content, 'utf-8')` -> `renameSync(BRAIN_STATE_TMP, BRAIN_STATE_FILE)`.

5. `function parseLLMJson(raw)` — Strip `<think>...</think>` blocks with regex `/<think>[\s\S]*?<\/think>/g`. Strip markdown fences `^```(?:json)?\s*` and trailing ` ```\s*$`. Try `JSON.parse(text)`. If fails, try regex `\{[\s\S]*\}` extraction and parse. Return parsed object or `null`.

6. `function buildBackgroundPrompt(recentHistory, gameState, existingState)` — Build the system prompt for background brain. Takes last 20 messages from `recentHistory` (slice(-20)), formats game state compactly, includes existing plan from `existingState`. Instructs model to output ONLY valid JSON with structure: `{ plan: { goal, steps: [{action, status}], current_step }, insights: [], spatial: [{label, note, hazard?}], constraints: [], updated_at }`. Includes `_agentName` in prompt.

7. `async function runBackgroundCycle(bot, config)` — Sets `_bgRunning = true` in try/finally. Reads `conversationHistory` from `mind/llm.js` via `getHistory()` (import at top). Gets compact game state from bot (pos, health, food, time, inventory count). Reads existing brain state from cache or file. Calls `bgClient.chat.completions.create()` with `model: BACKGROUND_MODEL`, `max_tokens: BACKGROUND_MAX_TOKENS`, `temperature: 0.3` (lower for structured output). Parses response with `parseLLMJson()`. If parse succeeds, merges new fields into existing state (preserving ring buffer continuity — append new insights to existing, don't replace). Calls `writeBrainState()`. Logs `[background-brain] cycle complete`. On any error, logs `[background-brain] cycle error: {message}` and does NOT throw.

8. `function formatBrainState(state)` — Format for prompt injection, capped at 1200 chars. Structure:
```
## Background Brain ({age}s ago)
Goal: {state.plan.goal}
Step: {current step action} ({status})
Constraints: {first 2 constraints joined by '. '}
Insights: {last 3 insights joined by ' '}
Hazards: {first 2 spatial entries where hazard=true, notes joined by '. '}
```
If `!state` return null. Calculate age from `Date.now() - state.updated_at`. Slice output to 1200 chars max.
