    ```javascript
    // gather.js — Collect N of a resource (find + navigate + dig loop)
    ```

    Imports:
    ```javascript
    import minecraftData from 'minecraft-data'
    import { navigateToBlock } from '../navigate.js'
    import { digBlock } from '../dig.js'
    import { normalizeBlockName } from '../normalizer.js'
    import { isInterrupted } from '../interrupt.js'
    ```

    Module-level: `const mcData = minecraftData('1.21.1')`

    Export `gather(bot, blockName, count = 1, options = {})`:

    Parameters:
    - `bot` — mineflayer bot instance
    - `blockName` — raw block name (will be normalized)
    - `count` — number of blocks to successfully dig (default 1)
    - `options.maxDistance` — search radius (default 64)
    - `options.navTimeout` — per-navigation timeout ms (default 30000)

    Implementation:
    1. Normalize: `const normalized = normalizeBlockName(blockName)`
    2. Look up block ID: `const blockId = mcData.blocksByName[normalized]?.id`
    3. If `blockId === undefined`, return `{ success: false, collected: 0, reason: 'unknown_block', blockName: normalized }`
    4. Initialize: `let collected = 0`
    5. Loop `while (collected < count)`:
       a. `if (isInterrupted(bot)) break` — check BEFORE work
       b. Find nearest block: `const candidates = bot.findBlocks({ matching: blockId, maxDistance: options.maxDistance || 64, count: 10 })`
       c. If `candidates.length === 0`, break — no more blocks in range
       d. Inner loop over candidates (try each until one succeeds or all exhausted):
          - `if (isInterrupted(bot)) break`
          - Get block object: `const target = bot.blockAt(candidates[i])`
          - If `!target || target.name === 'air'`, continue to next candidate
          - Equip best tool: `try { await bot.tool.equipForBlock(target) } catch {}`
          - `if (isInterrupted(bot)) break`
          - Navigate: `const nav = await navigateToBlock(bot, target, options.navTimeout || 30000)`
          - `if (isInterrupted(bot)) break`
          - If `!nav.success`, continue to next candidate (unreachable — skip, don't fail)
          - Re-fetch block (may have changed during nav): `const fresh = bot.blockAt(candidates[i])`
          - If `!fresh || fresh.name === 'air'`, continue
          - Dig: `const dig = await digBlock(bot, fresh)`
          - `if (isInterrupted(bot)) break`
          - If `dig.success`, increment `collected` and break inner loop (go find next)
       e. If inner loop exhausted all candidates without a successful dig, break outer loop (no reachable/diggable blocks left)
    6. Return `{ success: collected >= count, collected, requested: count, blockName: normalized }`

    CRITICAL: `bot.interrupt_code` check after EVERY `await` — this is the cooperative interrupt pattern. Do NOT skip any.

    Log progress to console: `console.log('[gather] ${collected}/${count} ${normalized}')` after each successful dig.
  </action>

    ```javascript
    // mine.js — Mine ore blocks with auto-best-tool selection
    ```

    Imports:
    ```javascript
    import minecraftData from 'minecraft-data'
    import { navigateToBlock } from '../navigate.js'
    import { digBlock } from '../dig.js'
    import { normalizeBlockName } from '../normalizer.js'
    import { isInterrupted } from '../interrupt.js'
    ```

    Module-level: `const mcData = minecraftData('1.21.1')`

    **Helper function** (not exported): `canHarvestWith(block, heldItemType)`
    - If `!block.harvestTools` return `true` (no restriction, any tool works)
    - If `heldItemType === null` return `false` (bare hands, but block requires a tool)
    - Return `!!block.harvestTools[heldItemType]`

    Export `mine(bot, oreName, count = 1, options = {})`:

    Parameters:
    - `bot` — mineflayer bot instance
    - `oreName` — raw ore block name (will be normalized, e.g., 'iron_ore', 'diamond_ore', 'coal_ore')
    - `count` — number of ore blocks to mine (default 1)
    - `options.maxDistance` — search radius (default 64)
    - `options.navTimeout` — per-navigation timeout ms (default 30000)

    Implementation:
    1. Normalize: `const normalized = normalizeBlockName(oreName)`
    2. Look up block ID: `const blockId = mcData.blocksByName[normalized]?.id`
    3. If `blockId === undefined`, return `{ success: false, mined: 0, reason: 'unknown_block', oreName: normalized }`
    4. Initialize: `let mined = 0`
    5. Loop `while (mined < count)`:
       a. `if (isInterrupted(bot)) break`
       b. Find ore blocks: `const candidates = bot.findBlocks({ matching: blockId, maxDistance: options.maxDistance || 64, count: 10 })`
       c. If empty, break
       d. Inner loop over candidates:
          - `if (isInterrupted(bot)) break`
          - Get block: `const target = bot.blockAt(candidates[i])`
          - If `!target || target.name === 'air'`, continue
          - Auto-select best tool: `try { await bot.tool.equipForBlock(target, { requireHarvest: true }) } catch {}`
          - After equip, CHECK if the held tool can actually harvest this ore:
            ```javascript
            const heldType = bot.heldItem ? bot.heldItem.type : null
            const harvestable = canHarvestWith(target, heldType)
            ```
          - If NOT harvestable, log warning `console.log('[mine] cannot harvest ${normalized} with current tools')` and return early:
            `return { success: false, mined, reason: 'no_suitable_tool', oreName: normalized }`
            (This is a hard stop — unlike gather, mining the wrong ore without the right tool tier is always wasteful)
          - `if (isInterrupted(bot)) break`
          - Navigate: `const nav = await navigateToBlock(bot, target, options.navTimeout || 30000)`
          - `if (isInterrupted(bot)) break`
          - If `!nav.success`, continue to next candidate
          - Re-fetch block: `const fresh = bot.blockAt(candidates[i])`
          - If `!fresh || fresh.name === 'air'`, continue
          - Dig: `const dig = await digBlock(bot, fresh)`
          - `if (isInterrupted(bot)) break`
          - If `dig.success`, increment `mined`
       e. If inner loop exhausted without success, break outer loop
    6. Return `{ success: mined >= count, mined, requested: count, oreName: normalized }`

    Log progress: `console.log('[mine] ${mined}/${count} ${normalized}')` after each successful dig.
