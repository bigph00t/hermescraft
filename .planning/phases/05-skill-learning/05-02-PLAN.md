<objective>
Add background reflection and experience-based skill creation to the planner loop. Every 5 minutes (~5 planner ticks at 60s interval), the planner does a deeper review: it asks the LLM to reflect on recent actions, extract lessons, and identify any new skills learned. Successful multi-step accomplishments are automatically saved as skills in SKILL.md format. Skills that consistently fail get downgraded.

Purpose: Agents get better over time without explicit programming. Reflection creates self-awareness; skill creation captures reusable knowledge; effectiveness tracking ensures only useful skills persist.

Output: Extended skills.js with createSkillFromExperience, extended planner.js with reflection cycle
</objective>

     ```
     Review the recent events above. Answer THREE questions concisely:

     1. WHAT WORKED: What actions or strategies succeeded? (1-2 sentences)
     2. WHAT FAILED: What went wrong or could be improved? (1-2 sentences)
     3. NEW SKILL: Did you accomplish something reusable that isn't already in your skill list? If yes, output EXACTLY this format on a new line:

     ```
