2b. In `summarizeState(state)` function, add a new section AFTER the "Recent chat" section (before `return lines.join('\n')`). Since ServerTap fetch is async but summarizeState is sync, use a cached approach:

Instead of making summarizeState async (would break all callers), add a module-level cache:

```javascript
let _serverSummaryCache = ''
let _serverSummaryLastFetch = 0
const SERVER_SUMMARY_INTERVAL = 60000  // refresh every 60s

async function refreshServerSummary() {
  try {
    const [info, players] = await Promise.all([getServerInfo(), getPlayers()])
    _serverSummaryCache = formatServerSummary(info, players)
    _serverSummaryLastFetch = Date.now()
  } catch {
    // Keep stale cache on failure
  }
}
```

Add a call to `refreshServerSummary()` in `fetchState()` (triggered every tick):
```javascript
// Refresh ServerTap cache every 60s
if (Date.now() - _serverSummaryLastFetch > SERVER_SUMMARY_INTERVAL) {
  refreshServerSummary()  // fire-and-forget, don't await
}
```

In `summarizeState()`, append the cached value:
```javascript
if (_serverSummaryCache) {
  lines.push(_serverSummaryCache)
```
