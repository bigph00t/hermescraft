Deploy the Skript scripts to the Paper server running in Docker and grant bot group permissions.

**Step 1: Identify the Docker container**
```bash
docker ps --filter "name=minecraft" --format '{{.Names}}'
```
The container is typically named `minecraft-server` or similar.

**Step 2: Copy Skript files to the server**
```bash
docker cp server/plugins/Skript/scripts/scan.sk CONTAINER:/data/plugins/Skript/scripts/
docker cp server/plugins/Skript/scripts/share-location.sk CONTAINER:/data/plugins/Skript/scripts/
docker cp server/plugins/Skript/scripts/myskills.sk CONTAINER:/data/plugins/Skript/scripts/
```

**Step 3: Reload Skript scripts**

Use ServerTap REST API (preferred) or RCON to reload:
```bash
# Via ServerTap (if port 4567 is exposed)
curl -X POST "http://localhost:4567/v1/server/exec" -H "Content-Type: application/json" -d '{"command": "sk reload scripts"}'

# Fallback via docker exec
docker exec CONTAINER rcon-cli "sk reload scripts"
```

**Step 4: Grant LuckPerms permissions to bot group**

The bot group (created in Phase 1) needs permission for the Skript commands:
```bash
# Via ServerTap or RCON
docker exec CONTAINER rcon-cli "lp group bot permission set skript.scan true"
docker exec CONTAINER rcon-cli "lp group bot permission set essentials.home true"
docker exec CONTAINER rcon-cli "lp group bot permission set essentials.sethome true"
docker exec CONTAINER rcon-cli "lp group bot permission set quickshop.create.sell true"
docker exec CONTAINER rcon-cli "lp group bot permission set quickshop.find true"
```

If rcon-cli is not available (per STATE.md note about ServerTap), use:
```bash
curl -X POST "http://localhost:4567/v1/server/exec" -H "Content-Type: application/json" -d '{"command": "lp group bot permission set skript.scan true"}'
curl -X POST "http://localhost:4567/v1/server/exec" -H "Content-Type: application/json" -d '{"command": "lp group bot permission set essentials.home true"}'
curl -X POST "http://localhost:4567/v1/server/exec" -H "Content-Type: application/json" -d '{"command": "lp group bot permission set essentials.sethome true"}'
curl -X POST "http://localhost:4567/v1/server/exec" -H "Content-Type: application/json" -d '{"command": "lp group bot permission set quickshop.create.sell true"}'
curl -X POST "http://localhost:4567/v1/server/exec" -H "Content-Type: application/json" -d '{"command": "lp group bot permission set quickshop.find true"}'
```

**Step 5: Verify scripts loaded**
```bash
docker exec CONTAINER rcon-cli "sk info"
```
Should show scan.sk, share-location.sk, myskills.sk as loaded scripts.
