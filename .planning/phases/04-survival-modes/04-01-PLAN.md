**Module-level constants:**
- `mcData = minecraftData('1.21.1')`
- `HOSTILE_MOBS = new Set(Object.values(mcData.entities).filter(e => e.category === 'Hostile mobs').map(e => e.name))` — builds the 42-entry hostile mob lookup Set at module load
- `COMBAT_RETREAT_HEALTH = 6` — retreat at 3 hearts
- `COMBAT_ATTACK_RANGE = 4` — blocks within which bot can melee attack
- `ATTACK_COOLDOWN_MS = 600` — 600ms between attacks (safe for unarmed; sword is faster but this is conservative)

**Export 1: `attackTarget(bot, target)`**
- Purpose: One attack action, called by the body tick (one hit per 300ms tick cycle)
- Guard: if `!target.isValid || target.health <= 0` return `{ success: false, reason: 'target_gone' }`
- Health retreat check: if `bot.health <= COMBAT_RETREAT_HEALTH`, compute flee direction as `bot.entity.position.minus(target.position).normalize().scaled(12).plus(bot.entity.position)`, call `navigateTo(bot, Math.floor(away.x), Math.floor(away.y), Math.floor(away.z), 1, 5000)`, return `{ success: false, reason: 'retreated_low_health' }`
- Distance check: if `target.position.distanceTo(bot.entity.position) > COMBAT_ATTACK_RANGE`, use `bot.pathfinder.setGoal(new goals.GoalFollow(target, 2), true)` (dynamic=true for moving target) and return `{ success: true, reason: 'closing_distance' }`
- In range: `bot.pathfinder.setGoal(null)` to stop movement, `await bot.lookAt(target.position.offset(0, target.height / 2, 0))`, `bot.attack(target)`, return `{ success: true }`
- Wrap body in try/catch; on error return `{ success: false, reason: err.message }`

**Export 2: `combatLoop(bot, target)`**
- Purpose: Sustained attack loop until target is dead or bot retreats. Used by LLM dispatch (`!combat`).
- While loop: `while (target.isValid && target.health > 0)`
  - Check `isInterrupted(bot)` — return `{ success: false, reason: 'interrupted' }` if set
  - Retreat check: if `bot.health <= COMBAT_RETREAT_HEALTH`, compute flee vector (same as attackTarget), navigate away with 5s timeout, return `{ success: false, reason: 'retreated_low_health' }`
  - Distance > COMBAT_ATTACK_RANGE: `bot.pathfinder.setGoal(new goals.GoalFollow(target, 2), true)`, `await new Promise(r => setTimeout(r, 200))`, continue
  - In range: `bot.pathfinder.setGoal(null)`, `await bot.lookAt(target.position.offset(0, target.height / 2, 0))`, `bot.attack(target)`, `await new Promise(r => setTimeout(r, ATTACK_COOLDOWN_MS))`
- After loop: return `{ success: true }`
- Wrap in try/catch; on error return `{ success: false, reason: err.message }`

**Also export `HOSTILE_MOBS` Set** so `body/modes.js` (Plan 02) can import it for hostile detection without rebuilding.

Follow project conventions: ES Modules, 2-space indent, no-semi, single quotes, no default exports, module header comment.
