**Module header and imports:**

```
// modes.js -- 300ms autonomous body tick: survival, combat, unstuck, pickup, idle
```
- Import `attackTarget, HOSTILE_MOBS` from `./skills/combat.js`

**Module-level constants:**
- `TICK_MS = 300`
- `FIRE_BLOCKS = new Set(['fire', 'soul_fire', 'lava', 'campfire', 'soul_campfire'])`
- `HAZARD_FLEE_HEALTH = 16` — flee environmental hazards when < 8 hearts
- `EAT_THRESHOLD = 14` — eat when food < 14 (sprint disabled)
- `STUCK_TICK_THRESHOLD = 10` — 10 x 300ms = 3 seconds
- `ITEM_PICKUP_RANGE = 8` — scan radius for dropped items
- `IDLE_LOOK_RANGE = 16` — scan radius for entities to look at
- `COMBAT_ENGAGE_RANGE = 8` — engage hostiles within 8 blocks

**Module-level mutable state:**
- `let _lastPos = null` (stuck detection)
- `let _stuckTicks = 0` (stuck detection)
- `let _lastLookTime = 0` (idle look cooldown — once per 3s, not every 300ms)
- `let _tickBusy = false` (guard against overlapping async ticks)

**Priority 1: `checkSurvival(bot, getSkillRunning)`** (always fires, even during skills)
- Auto-eat: if `bot.food < EAT_THRESHOLD`, call `eatIfHungry(bot, EAT_THRESHOLD)`. Gate: only call if `!getSkillRunning()` to avoid eating mid-combat. But if `bot.food <= 2`, eat even if skill running (starvation override).
- Hazard flee: call `isInHazard(bot)` — checks `bot.blockAt(bot.entity.position.floored())` for FIRE_BLOCKS, checks body-level block for lava, checks `bot.oxygenLevel <= 3` for drowning. If hazard detected AND a skill is running, call `requestInterrupt(bot)` first to cooperatively cancel the skill, then navigate to safety. For fire/lava: navigate 5 blocks in the opposite direction of the hazard (use bot's current velocity direction or just +Y for lava). For drowning: navigate to `bot.entity.position.offset(0, 5, 0)` (up). Use `navigateTo` with 5000ms timeout.
- Return true if any survival action was taken.

**Helper: `isInHazard(bot)`**
- `const feetPos = bot.entity.position.floored()`
- `const feetBlock = bot.blockAt(feetPos)` — check `FIRE_BLOCKS.has(feetBlock?.name)`
- `const bodyBlock = bot.blockAt(feetPos.offset(0, 1, 0))` — check lava
- `if (bot.oxygenLevel <= 3)` return `'drowning'`
- Return hazard type string or null

**Priority 2: `checkCombat(bot)`** (only if `!getSkillRunning()`)
- Find nearest hostile: `bot.nearestEntity(e => e.type === 'mob' && HOSTILE_MOBS.has(e.name) && e.position.distanceTo(bot.entity.position) < COMBAT_ENGAGE_RANGE)`
- If found, call `attackTarget(bot, target)` — single hit, non-blocking
- Return true if attack was attempted

**Priority 3: `checkStuck(bot)`** (only if `!getSkillRunning()`)
- Read `bot.entity.position` and `bot.pathfinder.isMoving()`
- If not moving: reset `_stuckTicks = 0`, `_lastPos = null`, return false
- If moving and `_lastPos` exists and `pos.distanceTo(_lastPos) < 0.5`: increment `_stuckTicks`
  - If `_stuckTicks >= STUCK_TICK_THRESHOLD`: `bot.pathfinder.setGoal(null)`, reset, log `[modes] stuck detected — halting pathfinder`, return true
- Else: `_stuckTicks = 0`
- `_lastPos = pos.clone()`
- Return false

**Priority 4: `checkItemPickup(bot)`** (only if `!getSkillRunning()` and `!bot.pathfinder.isMoving()`)
- Scan `Object.values(bot.entities)` for `e.name === 'item'` within `ITEM_PICKUP_RANGE` blocks (per Pitfall 3: use `entity.name`, NOT `entity.type`)
- Sort by distance ascending, pick closest
- If found: `navigateTo(bot, Math.floor(item.position.x), Math.floor(item.position.y), Math.floor(item.position.z), 1, 3000)` — 3s timeout, short range pickup
- Return true if pickup navigation started

**Priority 5: `checkIdleLook(bot)`** (only if `!getSkillRunning()` and `!bot.pathfinder.isMoving()`)
- Rate limit: if `Date.now() - _lastLookTime < 3000` return false (look at something every 3s max)
- Find a nearby entity: `bot.nearestEntity(e => e.position.distanceTo(bot.entity.position) < IDLE_LOOK_RANGE && e !== bot.entity)`
- If found: `await bot.lookAt(entity.position.offset(0, entity.height / 2, 0))`, update `_lastLookTime = Date.now()`
- Return true if looked

**The tick function:**
```javascript
async function bodyTick(bot, getSkillRunning) {
  if (_tickBusy) return  // previous tick still running
  _tickBusy = true
  try {
    if (await checkSurvival(bot, getSkillRunning)) return
    if (getSkillRunning()) return  // P2-P5 gated
    if (await checkCombat(bot)) return
    if (await checkStuck(bot)) return
    if (!bot.pathfinder.isMoving()) {
      if (await checkItemPickup(bot)) return
      await checkIdleLook(bot)
    }
  } catch (err) {
    // Body tick must never crash — log and continue
    console.error('[modes] tick error:', err.message)
  } finally {
    _tickBusy = false
  }
}
```

**`initModes(bot, getSkillRunning)` export:**
- Start the tick: `setInterval(() => bodyTick(bot, getSkillRunning), TICK_MS)`
- Log: `console.log('[modes] body tick started — 300ms interval')`
- Return the interval ID (so it can be cleared in tests or shutdown)

Follow project conventions: ES Modules, 2-space indent, no-semi, single quotes, no default exports, module header comment.
