   ```bash
       AGENT_NAME='name' \
       MC_USERNAME='name' \
       ... (all env vars with PLACEHOLDER pattern) ...
           node SCRIPT_DIR_PLACEHOLDER/start.js
       CODE=$?
       [ $CODE -eq 0 ] && break
       [ $CODE -eq 42 ] && { echo "[!] Name scheduled restart — relaunching..."; continue; }
       echo "[!] Name crashed (exit $CODE) — restarting in 5s..."
       sleep 5
   done
   ```

   ```
   Windows:
     luna  — Luna (artist, creative builder)
     max   — Max (engineer, infrastructure)
     ivy   — Ivy (naturalist, farms & gardens)
   ```
