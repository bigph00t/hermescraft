    4. Main export `async function smelt(bot, inputName, fuelName, count = 1, options = {})`:
       - Normalize both names: `normalizeItemName(inputName)`, `normalizeItemName(fuelName)`
       - Look up item IDs: `mcData.itemsByName[normalizedInput]?.id`, same for fuel
       - If either ID is undefined, return `{ success: false, reason: 'unknown_item', item: normalizedInput || normalizedFuel }`
       - Calculate timeout: `const timeoutMs = options.timeoutMs ?? (count * 12_000 + 5_000)` (12s per item + 5s buffer — each item takes 10s to smelt, plus safety margin)
       - Find furnace: search for each FURNACE_BLOCKS type using `bot.findBlock({ matching: mcData.blocksByName[name].id, maxDistance: 32 })`, take first found
       - If no furnace found, return `{ success: false, reason: 'no_furnace_nearby' }`
       - Navigate to furnace: `await navigateToBlock(bot, furnaceBlock)` — check success, check interrupt
       - Open furnace: `const furnace = await bot.openFurnace(furnaceBlock)`
       - Wrap remaining logic in try/finally with `furnace.close()` in finally block
       - Calculate fuel count: `Math.ceil(count / 8) + 1` (coal/charcoal smelts 8; extra 1 for safety with partial fuels)
       - Put fuel: `await furnace.putFuel(fuelId, null, fuelCount)` — check interrupt
       - Put input: `await furnace.putInput(inputId, null, count)` — check interrupt
       - Event-driven wait:
         ```javascript
         await new Promise((resolve, reject) => {
           const timer = setTimeout(() => reject(new Error('smelt_timeout')), timeoutMs)
           const check = () => {
             if (furnace.outputItem()) {
               clearTimeout(timer)
               furnace.removeListener('update', check)
               resolve()
             }
           }
           furnace.on('update', check)
           check() // immediate check in case already done
         })
         ```
       - Take output: `await furnace.takeOutput()`
       - Log: `console.log('[smelt] smelted ${count}x ${normalizedInput}')`
       - Return `{ success: true, item: normalizedInput, count }`
       - Catch smelt_timeout: return `{ success: false, reason: 'smelt_timeout', item: normalizedInput }`
       - Catch other errors: return `{ success: false, reason: err.message, item: normalizedInput }`
    5. On interrupt during smelt wait, the finally block closes the furnace. The interrupt check happens before and after the long wait.
