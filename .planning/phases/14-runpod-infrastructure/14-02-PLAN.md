   e. No "context length" or "maximum input length" errors in the first 20 ticks
   f. Agents interact with the MC world (mining, building, chatting)

   - Thinking tokens are eating the MAX_TOKENS budget
   - Try: edit .env.runpod, set MAX_TOKENS=256, restart agents

4. If tool call format looks wrong (JSON instead of !command):
