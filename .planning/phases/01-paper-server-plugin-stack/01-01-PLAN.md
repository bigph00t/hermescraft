  <action>
All commands run on Glass via SSH. Execute these steps in order:

1. Stop and remove the existing Fabric server container:
   ssh glass "docker stop mc-server 2>/dev/null; docker rm mc-server 2>/dev/null"

2. Back up the existing server directory:
   ssh glass "cp -r /opt/hermescraft/server/hermescraft-world /opt/hermescraft/server/hermescraft-world-backup-$(date +%Y%m%d)"

3. Download Paper 1.21.1 build #133:
   ssh glass "curl -L -o /opt/hermescraft/server/paper-1.21.1.jar 'https://api.papermc.io/v2/projects/paper/versions/1.21.1/builds/133/downloads/paper-1.21.1-133.jar'"

4. Verify the jar downloaded successfully:
   ssh glass "ls -la /opt/hermescraft/server/paper-1.21.1.jar"

5. Migrate world directories for Paper's dimension splitting.
   Paper expects:
   - hermescraft-world/region/ (overworld — stays as-is)
   - hermescraft-world_nether/DIM-1/region/ (nether — needs restructuring)
   - hermescraft-world_the_end/DIM1/region/ (end — needs restructuring)

   Check if nether/end exist in the current world:
   ssh glass "ls /opt/hermescraft/server/hermescraft-world/DIM-1/ 2>/dev/null; ls /opt/hermescraft/server/hermescraft-world/DIM1/ 2>/dev/null"

   If DIM-1 exists (nether):
   ssh glass "mkdir -p /opt/hermescraft/server/hermescraft-world_nether && mv /opt/hermescraft/server/hermescraft-world/DIM-1 /opt/hermescraft/server/hermescraft-world_nether/DIM-1"

   If DIM1 exists (end):
   ssh glass "mkdir -p /opt/hermescraft/server/hermescraft-world_the_end && mv /opt/hermescraft/server/hermescraft-world/DIM1 /opt/hermescraft/server/hermescraft-world_the_end/DIM1"

   If neither exists, that's fine — they'll be created when players visit those dimensions.

6. Ensure eula.txt is accepted:
   ssh glass "echo 'eula=true' > /opt/hermescraft/server/eula.txt"

7. Update server.properties — ensure peaceful mode, survival, correct world name:
   ssh glass "cat /opt/hermescraft/server/server.properties" — read current properties first.
   Then ensure these values are set (use sed or write the full file):
   - level-name=hermescraft-world
   - difficulty=peaceful
   - gamemode=survival
   - level-type=minecraft\:normal
   - online-mode=false (for bot access)
   - max-players=20

8. Start Paper server in Docker:
   ssh glass "docker run -d --name mc-server -p 25565:25565 --memory=3g -v /opt/hermescraft/server:/data -w /data eclipse-temurin:21-jre java -Xmx3G -Xms2G -jar paper-1.21.1.jar nogui"

9. Wait for server startup (30-60 seconds), then check logs:
   ssh glass "docker logs mc-server 2>&1 | tail -30"
   Look for "Done" message indicating server is ready.

10. Verify server.properties was preserved by Paper (Paper may regenerate it on first run):
    ssh glass "grep -E 'difficulty|gamemode|level-name' /opt/hermescraft/server/server.properties"
  </action>
