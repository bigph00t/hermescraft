  <action>
Create agent/needs.js with two exported functions:

**detectBehaviorMode(state)** — returns one of: "work", "shelter", "social", "sleep"
- Uses `state.time` (Minecraft ticks 0-24000):
  - 0-11500: "work" (dawn through late afternoon)
  - 11500-12500: "shelter" (dusk — get inside)
  - 12500-22000: "social" (night in shelter — chat, organize, reflect)
  - 22000-24000: "sleep" (late night — stay put, minimal activity)
- Returns "shelter" override if state.health < 8 regardless of time (survival priority)

**calculateNeeds(state, socialState)** — returns `{ hunger, safety, social, creative, priority }` where each is 0-100 and priority is the name of the highest need.
- `socialState` is an object `{ lastChatTimestamp, nearbyPlayerCount }` computed by the caller.
- **hunger** (0 = starving, 100 = full):
  - `Math.round((state.food / 20) * 100)` — direct mapping from food level
- **safety** (0 = danger, 100 = safe):
  - Start at 100
  - Subtract `(20 - state.health) * 3` for low health (0 health = -60)
  - Subtract 20 if any hostile mob within 16 blocks in nearbyEntities (check entity type for zombie, skeleton, spider, creeper, enderman, witch, drowned, phantom)
  - Subtract 15 if behavior mode is night AND agent is NOT within 20 blocks of home
  - Clamp to 0-100
- **social** (0 = lonely, 100 = fulfilled):
  - Start at 50
  - Add 30 if nearbyPlayerCount > 0
  - Add 20 if lastChatTimestamp is within the last 5 minutes (300000ms)
  - Subtract 5 for every 10 minutes since lastChatTimestamp (max penalty -50)
  - Clamp to 0-100
- **creative** (0 = bored, 100 = engaged):
  - Start at 70 (default baseline — agents generally feel "okay")
  - Subtract 3 per minute since last build_complete or routine action in recent events (caller can pass `lastActivityTimestamp`)
  - Clamp to 0-100
  - This is the "boredom" signal — drops when idle too long
- **priority**: the need with the LOWEST score (most urgent). Ties broken by: safety > hunger > social > creative.

Also export a helper:
**formatNeedsForPrompt(needs)** — returns a single-line string like: `"Needs: hunger=85(ok) safety=40(LOW) social=25(lonely) creative=60(ok) → Priority: safety"`
- Thresholds for labels: >= 70 "ok", 40-69 "moderate", < 40 "LOW"

Do NOT import from other modules — this is a pure calculation module. The caller (planner.js) passes all needed data as arguments.
